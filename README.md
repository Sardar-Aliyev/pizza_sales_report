# 🍕 Pizza Sales Analysis Dashboard

## Overview

This project analyzes pizza sales data using **Oracle SQL** and **Power BI** to uncover business insights and sales performance trends.

## Project Workflow

* Queried data in **Oracle SQL** to calculate key business metrics.
* Connected **Power BI** directly to the Oracle database.
* Built an interactive dashboard based on SQL queries.

## Key KPIs

* Total Revenue
* Total Orders
* Total Quantity Sold
* Average Pizzas per Order
* Average Revenue per Order

## Dashboard Insights

* Sales by Pizza Category
* Yearly Sales Trend
* Best-Selling Pizzas
* Lowest-Selling Pizzas

## Tools Used

* Oracle SQL
* Power BI

## Skills Demonstrated

* SQL Query Writing
* KPI Development
* Data Visualization
* Business Intelligence
* Sales Performance Analysis
